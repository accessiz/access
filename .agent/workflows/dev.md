// turbo-all

Este workflow tiene permisos para auto-ejecutar comandos seguros.

## Comandos comunes

1. Verificar lint
```bash
npm run lint
```

2. Build de producción
```bash
npm run build
```

3. Iniciar servidor de desarrollo
```bash
npm run dev
```

4. Verificar tipos TypeScript
```bash
npx tsc --noEmit
```

## Operaciones de archivos

5. Listar archivos
```bash
ls
```

6. Ver contenido de directorio
```bash
dir
```

## Git (solo lectura)

7. Ver estado de git
```bash
git status
```

8. Ver historial reciente
```bash
git log -n 5 --oneline
```
