1. Introducción

El Problema: Las agencias de modelos pequeñas y medianas manejan operaciones complejas que requieren agilidad y profesionalismo. Actualmente, dependen de herramientas fragmentadas como hojas de cálculo, correos electrónicos, PDFs manuales y WhatsApp para coordinar castings, presentar modelos y gestionar horarios. Este enfoque es ineficiente, propenso a errores, consume tiempo y proyecta una imagen poco profesional ante clientes de alto nivel.

La Solución (NYXA): Es una plataforma web moderna, intuitiva y centralizada, diseñada específicamente para las necesidades de las agencias de modelos. Reemplaza el caos con un flujo de trabajo optimizado, permitiendo a los bookers centrarse en descubrir talento y conseguir trabajos. Se desarrollará como un sistema "marca blanca" (white-label), adaptable para diferentes agencias, comenzando con una implementación a medida para IZ Management bajo el nombre "IZ Access".

2. Planes y Funcionalidades

Se ofrecerá un modelo de suscripción escalonado. El plan gratuito estará limitado a 30 días.

4. Plan de Trabajo

El desarrollo se dividirá en dos fases estratégicas, con un enfoque iterativo y pruebas con usuarios reales para validar la experiencia.

Fase 1 MVP para IZ Management

Paso 1: Diseño UI/UX y Configuración del Proyecto

Paso 2: Landing Page y Autenticación

Paso 3: Dashboard y Gestión de Modelos (CRUD)

Paso 4: Funcionalidad Core (Castings y Comp Cards)

Paso 5: Pruebas y Despliegue

Fase 2 Expansión a SaaS Multi-Agencia

Paso 1: Adaptación a Multi-Tenant

Paso 2: Funcionalidades Pro y Enterprise

Paso 3: Internacionalización y Escalabilidad
